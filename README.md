 Excited to Share My Latest Data Science Project: Heart Stroke Prediction 

🌟I'm thrilled to present my recent work on building a machine learning model to predict the risk of heart stroke using a healthcare dataset. This project combines data preprocessing, exploratory data analysis, and predictive modeling to uncover meaningful insights and deliver actionable results.🔍 Project Overview:The goal was to predict the likelihood of a stroke based on factors like age, hypertension, heart disease, BMI, smoking status, and more. I used a Logistic Regression model to classify stroke risk, achieving an accuracy of 58.33% and an F1 score of 0.51, indicating a balanced performance for this challenging binary classification task.💻 Key Steps & Techniques:✅ Data Preprocessing: Handled missing BMI values using mode imputation and encoded categorical variables (e.g., gender, smoking status) for model compatibility.✅ Exploratory Data Analysis (EDA): Visualized correlations and trends, revealing surprising insights—like non-smokers showing higher stroke risk in some cases, warranting further investigation.✅ Feature Engineering: Standardized features using StandardScaler to ensure robust model performance.✅ Model Evaluation: Assessed performance with metrics like accuracy, F1 score, mean squared error, and confusion matrix visualization using Seaborn.✅ Interactive Prediction: Built a user-friendly Stroke Risk Calculator that outputs risk probability and categorizes it as Low, Moderate, or High based on user inputs.🛠 Tech Stack:Languages: PythonLibraries: Pandas, NumPy, Scikit-learn, Matplotlib, SeabornTools: Jupyter Notebook📊 Key Insights:Age, hypertension, and heart disease strongly influence stroke risk, but surprisingly, lower hypertension and heart disease cases showed elevated stroke probabilities in some instances—an area for deeper analysis.Non-smokers exhibited higher stroke risk than expected, challenging common assumptions.BMI (20–50 range), marital status, residence type, and work type also impacted stroke likelihood, highlighting the complexity of health predictions.🎯 Impact & Learnings:This project deepened my understanding of handling imbalanced datasets and the importance of feature scaling in classification tasks. It also underscored the need for thorough EDA to uncover counterintuitive patterns. While the model provides a solid baseline, future iterations could explore advanced algorithms like Random Forest or XGBoost to boost accuracy.
